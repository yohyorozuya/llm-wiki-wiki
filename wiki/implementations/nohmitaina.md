# nohmitaina (nowissan)

## Overview

A macOS desktop editor for the [[llm-wiki-pattern]] that works with [[claude-code|Claude Code]] or Codex CLI over local markdown (no API key required). Its outsized contribution to the field is conceptual: nowissan articulated the [[three-failure-modes]] (Identity, Level, Relationship) cleanly, and proposed the **Dream cycle** — a background consolidation pass borrowed from sleep-based human memory consolidation, emitting DDD-style events (DuplicateCandidateDetected, ConceptsMerged, ConceptRelationshipTyped, ConceptLevelChanged).
